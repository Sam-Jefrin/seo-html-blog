# SEO-HTML-Blog

A structurally optimized, content-driven blog platform built from scratch using pure HTML5. This project demonstrates core on-page SEO principles and semantic web architecture required for entry-level digital marketing and web content roles.

## Key Features & SEO Practices Implemented

* **Structured Tags**: Utilized tags like `<main>`, `<section>`, and `<article>` to ensure clean content isolation for search engine web crawlers.
* **Heading Hierarchy**: Implemented a strict heading flow (`<h1>` through `<h3>`) to map content priorities and improve indexing readability.
* **Accessible Navigation**: Designed clean, text-based navigation menus anchor-linked to page sections.
* **Contact Optimization**: Used the semantic `<address>` container with click-to-call (`tel:`) and click-to-email (`mailto:`) protocols.

## Practical Skills Demonstrated

This project demonstrates readiness for tasks involving:
* Content Management Systems (CMS) data uploading.
* On-page SEO auditing and meta-tag updates.
* Digital marketing data entry and web formatting.
